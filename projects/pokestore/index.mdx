Fake store for purchasing pokemon related items like pokeballs, medicines and items.

Features full product catalogue which searching and filtering of pokemon items. Cart system to add and remove items you would
like to purchase and and checkout system to confirm your purchase. User login to allow personalization such as showing past
purhcases and stats. Utilizes Redis for implementing token blacklisting for authentication scheme utilizing client-side sessions
with JWT.
